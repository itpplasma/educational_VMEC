309 ! (with parameter argument m, not modulus argument k)

311 ! The coding below implements an approximate inverse of the function
312 ! d(kappa) = ctwiddle(kappa) / sqrt(kappa)

314 ! kappa = 1 + (pi^2/8) * (d^2+sqrt(d^4-1)-1)
